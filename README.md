# Bibliotecas

Nesse projeto foi utilizado a biblioteca readline, para ler entradas do usuário
através do terminal.

Para instalar, digite no terminal:
`npm init -y`<br/>
`npm install readline-sync`

Para rodar um arquivo, digite no terminal:
`node nomeArquivo.js`
